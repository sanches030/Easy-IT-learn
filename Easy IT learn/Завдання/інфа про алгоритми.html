Умови Python і оператори If

Python підтримує звичайні логічні умови з математики:

Дорівнює: a == b

Не дорівнює: a != b

Менше ніж: a < b

Менше або дорівнює: a <= b

Більше ніж: a > b

Більше або дорівнює: a >= b

Ці умови можна використовувати кількома способами, найчастіше в операторах if і циклах. «Інструкція if» записується за допомогою ключового слова if .

Python покладається на відступи (пробіл на початку рядка), щоб визначити область у коді. Інші мови програмування часто використовують для цієї мети фігурні дужки.

Ключове слово elif — це спосіб Python сказати: «якщо попередні умови не відповідають дійсності, спробуйте цю умову».

Якщо у вас є тільки один оператор для виконання, ви можете розмістити його в тому ж рядку, що і оператор if.

Якщо у вас є тільки один оператор для виконання, один для if і один для else, ви можете розмістити все це в одному рядку:

Один рядок if else:

a = 2

b = 330

print("A")

if a < b

else:

print("B")

Ви також можете мати кілька операторів else в одному рядку:

Один рядок оператора if else із 3 умовами:

a = 330

b = 330

print("A")

if a > b

else:

print("=")

if a == b

else:

print("B")

Ключове слово and є логічним оператором і використовується для комбінування умовних операторів:

Перевірте, чи aбільше b, І якщо c більше a:

a = 200

b = 33

c = 500

if a > b and c > a:

print("Both conditions are True")

Ключове orслово є логічним оператором і використовується для комбінування умовних операторів:

Перевірте, якщо aбільше ніж b, АБО якщо a більше ніж c:

a = 200

b = 33

c = 500

if a > b or a > c:

print("At least one of the conditions is True")

Ключове not слово є логічним оператором і використовується для зворотного результату умовного оператора

Перевірте, якщо а НЕ більше ніж b:

a = 33

b = 200

if not a > b:

print("a NOT greater than b")

Ви можете мати ifоператори всередині ifоператорів, це називається вкладеними if операторами.

x = 41

if x > 10:

print("Above ten, ")

else:

print("but not above 20.")
